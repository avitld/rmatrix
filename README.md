# rmatrix
trashy and stupidly simple matrix rain effect in ruby
***
## fun stuff wooo!!:
### keybindings
``
0 - Speed set to 10 (the lower, the faster) \
1 - Speed set to 50
2 - Speed set to 200
3 - Speed set to 300
4 - Speed set to 400
5 - Speed set to 600
6 - Speed set to 700
7 - Speed set to 800
8 - Speed set to 700
9 - Speed set to 900
p - Pauses the effect
``
